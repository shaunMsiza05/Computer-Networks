- **Encapsulation Protocol**: VXLAN wraps Layer 2 Ethernet frames inside Layer 3 UDP packets using **MAC-in-UDP** encapsulation (UDP port 4789).
- **Overlay Network**: It creates a virtual Layer 2 network over a physical Layer 3 network‚Äîideal for multi-tenant data centers and cloud platforms.

### üß† Why VXLAN Was Invented
Traditional VLANs hit scaling walls:
- **Limited VLAN IDs** (only 4094)
- **Spanning Tree Protocol** blocks redundant paths
- **MAC table bloat** in large Layer 2 domains

VXLAN solves these by:
- **Using IP routing** to move Layer 2 traffic
- **Enabling multipath forwarding** with ECMP (Equal-Cost Multi-Path)
- **Supporting massive scalability** for virtualized workloads
