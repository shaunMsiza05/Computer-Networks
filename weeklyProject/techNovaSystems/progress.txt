## 📡 DHCP Pool Configurations

### 🔸 Staff LAN – `10.0.10.0/24`
```plaintext
ip dhcp excluded-address 10.0.10.1 10.0.10.29

ip dhcp pool STAFF-LAN
 network 10.0.10.0 255.255.255.0
 default-router 10.0.10.1
 dns-server 8.8.8.8
 lease 7
```
- **Start IP for dynamic assignment**: `10.0.10.30`
- **Serves**: Wired staff clients and wireless clients via FAT APs

### 🔸 Guest LAN – `10.0.11.0/24`
```plaintext
ip dhcp excluded-address 10.0.11.1 10.0.11.29

ip dhcp pool GUEST-LAN
 network 10.0.11.0 255.255.255.0
 default-router 10.0.11.1
 dns-server 8.8.8.8
 lease 1
```
- **Short lease** for high-turnover guest clients
- **FIT APs** and WLC operate in this subnet

## 🔐 ACL Strategy – DHCP Isolation

### 🔸 Prevent Guest DHCP Offers from Reaching Staff LAN
```plaintext
access-list 100 deny udp host 10.0.11.2 any eq 67
access-list 100 permit ip any any

interface vlan 10
 ip access-group 100 in
```

### 🔸 Prevent Staff DHCP Offers from Reaching Guest LAN
```plaintext
access-list 101 deny udp host 10.0.10.2 any eq 67
access-list 101 permit ip any any

interface vlan 11
 ip access-group 101 in
```
